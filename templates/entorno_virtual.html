Entorno Virtual

El entorno virtual en programación, especialmente en Python, es un espacio aislado que permite gestionar dependencias y paquetes de forma independiente del sistema global.

¿Cómo funciona?

Un entorno virtual permite crear un espacio aislado para desarrollar proyectos en Python (o en otros lenguajes) sin conflictos entre dependencias, facilitando así la gestión y el mantenimiento del código.

Comandos

- python -m venv (nombre): Crea un entorno virtual con un nombre específico.

- (nombre)\Scripts\activate: Activa el entorno virtual en la terminal.

- python app.py: Ejecuta el código Python y devuelve la página.

¿Cuándo utilizarlo?

Utiliza un entorno virtual siempre que necesites gestionar dependencias específicas, trabajar en múltiples proyectos, realizar pruebas, colaborar con otros o mantener la organización de tus proyectos.
